1) Importare il progetto

- Dal Package Explorer aprire il progetto appena importato ed eseguire il file Main.java presente sotto Tree-CSP-Solver\src\exec_pack
- Aperto il file Main.java eseguirlo premendo Run (Ctrl+F11)
